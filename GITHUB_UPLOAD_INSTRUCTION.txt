Загрузка v10 на GitHub Pages

1. Распакуйте архив prizhivetsya_public_v10.zip.
2. Откройте папку prizhivetsya_public_v10.
3. Выделите всё содержимое папки, не саму папку.
4. В репозитории GitHub откройте Add file → Upload files.
5. Перетащите выделенные файлы и папки.
6. Нажмите Commit changes.
7. Через 1–10 минут откройте https://ramanich92.github.io/prizhivetsya/

Файлов меньше 100, чтобы GitHub web-upload не ругался.
