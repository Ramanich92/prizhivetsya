# Приживётся?

Бесплатный информационный справочник по плодовым культурам, сортам и рискам выращивания по регионам.

## Текущий статус

- пилотный регион: Московская область;
- коммерческих разделов нет;
- саженцы не продаются;
- заявки не собираются;
- рекомендации не являются гарантией приживаемости или урожая.

- `index.html` — главная;
- `regions.html` — регионы;
- `region-moskovskaya-oblast.html` — пилотный регион;
- `start-here.html` — с чего начать;

- `sources.html` — источники;
