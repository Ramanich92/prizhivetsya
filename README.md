# Приживётся?

Бесплатный информационный справочник по плодовым культурам, ягодникам, сортам и рискам выращивания.

Текущий фокус: Московская область, культуры, сорта, подзоны, участок, источники и методика оценки риска.

## v23

Это преддоменная контентная версия: добавлены короткие страницы для читателя, антисписок рискованных покупок, маршрут первого сада и чеклист готовности к домену.

Проект не продаёт саженцы, не принимает заявки, не размещает рекламу питомников и не оказывает коммерческие консультации.
