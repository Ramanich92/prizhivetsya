# Приживётся? — public v13 no monetization

Статическая версия сайта для GitHub Pages без коммерческих блоков: без коммерческих страниц, форм и платных разделов.

Внутри остаётся бесплатный справочник:

- главная страница;
- Московская область;
- культуры;
- сорта;
- проверка участка;
- источники;
- о проекте;
- юридические черновики.

## Как загрузить на GitHub Pages

Загружайте содержимое папки в корень репозитория. Для настоящей чистки старых файлов лучше удалить из репозитория старые страницы: старые коммерческие страницы и файлы формы.
