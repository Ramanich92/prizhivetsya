# Приживётся?

Бесплатный информационный справочник по плодовым культурам, ягодникам, сортам и рискам выращивания.

Текущий пилот: Московская область.

В версии v17:
- коммерческие страницы и формы отсутствуют;
- удалены заготовки аналитики и cookie-баннер;
- добавлены страницы регионов, статуса данных, редакционной политики и исправлений;
- страница проверки участка называется `site-check.html`;
- навигация унифицирована.
