# Приживётся?

Бесплатный информационный справочник по плодовым культурам, ягодникам, сортам и рискам выращивания.

Текущий фокус: Московская область, методика оценки, проверка сортовой базы и честные оговорки.

В версии v21 добавлено ядро знаний: матрица культур, подзоны Подмосковья, болезни, подвои и процесс проверки сортов.
