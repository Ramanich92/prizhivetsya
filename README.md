# Приживётся? — публичная версия MVP

Статический сайт для проверки идеи: подбор плодовых культур и сортов по региону на примере Московской области.

Основные страницы: главная, Московская область, сорта, карточки сортов, план сада, заявка, источники и юридические черновики.

Важно: сортовая база пока является проверочным MVP-слоем. Рекомендации не являются гарантией приживаемости, урожая или успешной перезимовки.
