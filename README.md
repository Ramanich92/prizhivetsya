# Приживётся — v49

Региональный справочник по плодовым культурам, участкам, почве, схемам посадки, календарю, защите сада и покупке саженцев.

## Что изменено в v49

- синхронизированы `index.html`, `version.html`, `deployment-check.html` и `data/site-version.json`;
- убраны черновые метки версии из пользовательских страниц;
- сохранены разделы: регионы, подзоны, места, связки, схемы, календарь, защита, почва, покупка;
- главный следующий этап — редакционная вычитка, контакты, юридические тексты и проверка источников.

## После заливки

Откройте:
- `/index.html`
- `/version.html`
- `/deployment-check.html`
- `/data/site-version.json`

Если `version.html` показывает v49, а главная старую версию — не перезаписался `index.html` или GitHub Pages отдаёт кеш.
