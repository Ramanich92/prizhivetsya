# Приживётся? — v15 information edition

Бесплатный информационный справочник по плодовым культурам, ягодникам, сортам и рискам выращивания.

## Что изменилось в v15

- Убрана коммерческая логика.
- Добавлены информационные страницы: как пользоваться, методика оценки, термины, опылители, причины неприживаемости, выбор саженца, работа с источниками.
- Главная переписана под бесплатный справочник.
- Навигация обновлена под информационный проект.

## Публикация

Загружайте содержимое папки в корень репозитория GitHub Pages. Не загружайте саму папку как вложенную директорию.
