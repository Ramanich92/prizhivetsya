# Приживётся?

Бесплатный информационный справочник по плодовым культурам, ягодникам, сортам и рискам выращивания.

Текущий фокус: Московская область, культура → сорт → участок → источники.

## v22 information architecture

- единое меню на всех страницах;
- отдельный раздел «База знаний»;
- статическая страница сортов без зависимости от JavaScript;
- отдельный индекс культур;
- усиленная страница Московской области;
- улучшенные страницы культур;
- карта сайта;
- без монетизации, заявок, рекламы и платных разделов.
