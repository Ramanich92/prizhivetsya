# Приживётся v37

Региональный справочник по плодовым культурам, сортам и рискам выращивания.

В v37 добавлен третий слой: не только регион и подзона, но и ближайший районный или городской садовый ориентир. Это особенно важно для Московской области, Краснодарского края, Ростовской области, Ленинградской области, Урала и Башкортостана.
