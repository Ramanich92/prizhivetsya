# Приживётся?

Бесплатный информационный справочник по плодовым культурам, ягодникам, сортам и рискам выращивания по регионам.

Текущий фокус: Московская область. Проект не продаёт саженцы, не размещает рекламу питомников и не оказывает коммерческие консультации.

## Структура

- `index.html` — главная страница.
- `regions.html` — список регионов.
- `region-moskovskaya-oblast.html` — пилотная региональная страница.
- `varieties.html` и `varieties/` — список и карточки сортов.
- `site-check.html` — проверка условий участка.
- `methodology.html` — методика оценки риска.
- `sources.html` — источники и подход к проверке данных.

## Принцип

Справочник не гарантирует приживаемость или урожай. Он помогает снизить риск ошибки при выборе культуры, сорта и места посадки.
