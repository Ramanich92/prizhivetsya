Жимолость в Московской области

Надёжно

Один из самых удачных ягодных кустарников для прохладного климата. Для урожая нужны несколько сортов рядом.

Короткий вывод

Сажать 2–4 разных сорта рядом. Один куст часто даёт слабый результат.

Основные риски

  • один куст
  • сорта с горчинкой
  • осыпаемость ягод
  • пересыхание

Сорта-кандидаты

Список не является гарантией пригодности сорта. Каждая карточка постепенно проверяется по источникам.
