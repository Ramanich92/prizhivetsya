Жимолость

Жимолость

Надёжно

Один из самых удачных ягодных кустарников для прохладного климата. Для урожая нужны несколько сортов рядом.

Совет: Сажайте группой из 2–4 сортов: у жимолости перекрёстное опыление обычно заметно влияет на урожай.

Основные риски

  • один куст
  • сорта с горчинкой
  • осыпаемость
  • пересыхание в первые годы
