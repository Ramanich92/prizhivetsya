# Changelog

## v22 information architecture

- Унифицирована навигация на всех публичных страницах.
- Добавлены страницы: База знаний, Карта сайта, индекс культур.
- Страница сортов получила статические таблицы для SEO и доступности.
- Страница Московской области пересобрана с полноценной таблицей культур.
- Страницы культур стали более информативными и связаны с сортовыми карточками.
- Уточнена иерархия источников.
- Коммерция не возвращалась: нет заявок, тарифов, платных планов, рекламы и партнёрских блоков.
